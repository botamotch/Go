# コマンドまとめ

```
$ go mod init xxx  # プロジェクトの初期化
$ go get xxx       # プロジェクト内のモジュールのインストール（go.modを編集）
$ go mod tidy      # ソースコードと比較してモジュールを整理する
$ go run xxx.go    # プログラムの実行

$ go install xxx   # ツールをグローバルでインストール
```
